Managed Shovel

External shovel mechanism for RabbitMQ

Download this project as a .zip file Download this project as a tar.gz file

Managed Shovel is an external implementation of the shovel addin built into RabbitMQ.

It currently provides the same features in addition to the ability to limit the maximum number of hops a messages is allowed to do before it is discarded. This is especially useful to set up symmetric brokers where messages sent on one are sent to the other and vice-versa. With the built-in addin this scenario is not supported and messaged would go into an infinite loop.

Sample usage can be found in the demo.

Packaged binaries can be downloaded in the artifacts section of the TeamCity CI server, kindly hosted by the guys at CodeBetter.
